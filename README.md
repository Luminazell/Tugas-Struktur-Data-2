🚀 Features

Program memiliki 5 fitur utama:

1️⃣ Deduplication

Menghapus elemen duplikat dari list tanpa mengubah urutan awal.

2️⃣ Intersection of Two Arrays

Menampilkan elemen yang muncul di kedua array.

3️⃣ Anagram Checker

Mengecek apakah dua string merupakan anagram.

4️⃣ First Recurring Character

Menemukan karakter pertama yang muncul lebih dari satu kali dalam string.

5️⃣ Phone Book Simulation

Simulasi buku telepon sederhana dengan fitur:

Tambah kontak

Cari kontak

Tampilkan semua kontak

🛠️ Tech Stack

Python 3.x

Built-in library:

collections.OrderedDict

Tidak memerlukan library tambahan.
